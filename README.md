# gulp-build-v.2

Install:

npm i;

Start for development:

npm run dev

Project build:

npm run build

Compile the project into a zip archive:

npm run zip

Upload project to ftp server:

npm run deploy

Collect all svg icons into one file:

npm run svgSprive
